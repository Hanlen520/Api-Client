## 简介
python3编写的web api测试工具。使用官方库，安装完python环境包可以直接使用。


## 功能
- 请求
 - [X] get
 - [X] post
 - [X] header
 - [ ] put
 - [ ] delete

## 特点
 - 允许输入的url带有中文
 - 返回的结果如果是json，格式化输出
 - 返回的结果开头出现BOM、结束出现空格和回车时，提出警告
